# Vui Chơi Việt - Trò Chơi Dân Gian

Dự án giới thiệu các trò chơi dân gian Việt Nam, giúp khơi nguồn ký ức tuổi thơ và gắn kết tình thân.

## Giới thiệu
Trang web cung cấp thông tin chi tiết về các trò chơi dân gian như:
- Ô ăn quan
- Kéo co
- Rồng rắn lên mây
- Và nhiều trò chơi khác...

## Công nghệ sử dụng
- HTML5
- CSS3 (Vanilla CSS)
- Font Awesome (Icons)

## Cách chạy dự án
Chỉ cần mở file `index.html` trong trình duyệt của bạn.
